# PathTracerWeb

一个基于Canvas的软件路径追踪实现, 纯TS实现，支持BVH加速，支持多种材质和纹理贴图

## 渲染效果

采样次数32, 递归深度6:

## 使用

### 安装依赖包

```bash
# npm
npm install 

# pnpm
pnpm install

# bun
bun install
```

### 运行

```bash
# npm
npm run dev

# pnpm
pnpm dev

# bun
bun dev
```
